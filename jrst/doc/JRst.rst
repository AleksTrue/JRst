Jrst
====

JRst est un parser RestructuredText en Java.

 - une liste
 - un autre element

 o une autre liste
 o encore un element
 o et un 3eme

Il permet d'ajouter des extenstions très facilement en construisant deux
classe. Une pour parser l'élément, l'autre qui représente l'élément parsé.

   + pour le plasir
   + pour verifier
